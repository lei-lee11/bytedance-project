# Express Demo App

A simple Express.js task management app.

## Installation

1. Clone the repository
2. Navigate to the project directory: `cd express-demo-app`
3. Install dependencies: `npm install`

## Running the App

1. Ensure the .env file has PORT=3000 (already configured)
2. Start the server: `npm start`
3. Visit http://localhost:3000 in your browser

## API Endpoints

- GET /: Returns a welcome message
- GET /api/tasks: Returns a list of sample tasks
